一、字彙選擇題 (Vocabulary) - 點擊括號看答案
1. ( B ) I want to buy a book. I am going to the ______.
(A) bakery (B) bookstore (C) hospital
2. ( A ) My father is a doctor. He goes to the ______ every day.
(A) hospital (B) park (C) supermarket
3. ( C ) We can play and ride a bike in the ______.
(A) post office (B) restaurant (C) park
4. ( B ) Mom wants to buy some vegetables. She is going to the ______.
(A) bank (B) supermarket (C) bakery
5. ( C ) I am hungry. I want to eat something. Let's go to the ______.
(A) post office (B) hospital (C) restaurant
二、發音測驗 (Phonics)
1. ( B ) 哪一個單字含有與「black」相同的發音？
(A) clay (B) block (C) glad
2. ( A ) 哪一個單字含有與「plane」相同的發音？
(A) plant (B) glass (C) clean
3. ( C ) 哪一個單字含有「gl」的發音？
(A) clock (B) play (C) glass
三、文法選擇題 (Grammar)
1. ( C ) Where ______ you going?
(A) is (B) am (C) are
2. ( A ) He ______ going to the bank.
(A) is (B) am (C) are
3. ( A ) A: Are you going to the bakery? B: No, I'm ______.
(A) not (B) isn't (C) aren't
四、閱讀測驗 (Reading Comprehension)
Today is Saturday. Kevin and his dog, Lucky, are going to the park.
Mr. Lin is going to the supermarket.
Ms. Green is a doctor. She is going to the hospital.
1. ( B ) Where is Kevin going?
(A) supermarket (B) park (C) hospital
2. ( C ) Who is going to the hospital?
(A) Kevin (B) Mr. Lin (C) Ms. Green
五、句子重組 (Unscramble) - 點擊橫線看答案
1. going / I / am / park / to / the / .
I am going to the park.
2. he / going / Where / is / ?
Where is he going?
3. to / are / the / going / bookstore / They / .
They are going to the bookstore.
